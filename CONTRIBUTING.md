# 🤝 Cómo contribuir

¡Gracias por tu interés en mejorar Subtitulador Fácil!

## 🐛 Reportar errores
Abre un **issue** en GitHub describiendo el problema y cómo reproducirlo.

## ✨ Sugerir mejoras
Abre un **issue** con la etiqueta `enhancement`.

## 🔧 Enviar código
1. Haz un fork del repositorio
2. Crea una rama (`git checkout -b mi-mejora`)
3. Haz commit de tus cambios (`git commit -m "Añadida X funcionalidad"`)
4. Haz push a la rama (`git push origin mi-mejora`)
5. Abre un Pull Request

## 🌍 Añadir idiomas
Si quieres traducir la interfaz a otro idioma, copia el bloque de traducciones y añade el tuyo en `app/webui.py`.

¡Toda ayuda es bienvenida!
